Train Schedule: A Firebase Usage App

This is a proof of concept for utilizing Google Firebase, the ready-made server technology, as well as using Moment.js for time and date parsing. Simple, effective design that works in a basic way.

As I am presenting most of these demos as-is, if this was a project completed to full satisfaction, I'd add a way to delete trains--as well as some form validation.
